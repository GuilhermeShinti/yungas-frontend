
## Iniciando o projeto
----------
```
# Clone este repositório
git clone

# Acesse o diretório do projeto
cd

# Para subir as aplicações para o docker, utilize o comando:
docker-compose up -d

# Para encerrar as aplicações:
docker-compose down
```
